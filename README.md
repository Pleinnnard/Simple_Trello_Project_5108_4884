Mini Integradora - ANÁLISIS Y DISEÑO DE SOFTWARE CON APLICACIONES WEB

Proyecto: Tello Básico

Este proyecto es un gestor de tareas tipo Kanban (estilo Trello) que permite administrar el flujo de trabajo a través de diferentes estados utilizando una arquitectura de capas y persistencia de datos en formato JSON.

Funcionalidades
*   **Tablero Kanban:** Visualización de tareas en columnas (To Do, Doing, Done).
*   **Gestión de Tareas:** Crear, actualizar y eliminar tareas (CRUD completo).
*   **Drag & Drop:** Movimiento intuitivo de tareas entre columnas arrastrando las tarjetas.
*   **Persistencia:** Los datos se guardan automáticamente en un archivo `tasks.json`.

Stack Tecnológico
*   **Backend:** Node.js, Express.js.
*   **Frontend:** HTML5, CSS3, JavaScript Vanilla, Bootstrap 5.
*   **Comunicación:** API REST y Fetch API.

Estructura del Proyecto (Backend)
*   **Controllers:** Manejo de peticiones y respuestas HTTP.
*   **Services:** Lógica de negocio y validaciones.
*   **Repository:** Acceso y manipulación del archivo de datos JSON.
*   **Routes:** Definición de los endpoints de la API.
